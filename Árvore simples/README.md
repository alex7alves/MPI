# Árvore simples

Implementação de uma árvore simples em MPI. A árvore aqui implementada pode ser visualizada na figura a seguir: 

![](https://github.com/alex7alves/Imagens/blob/master/%C3%81rvore.png)




## Como compila ?
	mpicc -o arvore arvore.c


## Como executa ?
	mpiexec  -n  8  ./arvore
